# House Price Prediction using Decision Trees

## Overview

This project explores the use of Decision Tree Regression to predict house prices using the California Housing dataset. The project follows a complete machine learning workflow, from loading and exploring the data to training, evaluating, and improving a predictive model.

The notebook was completed while studying machine learning concepts through Kaggle Learn and serves as one of my first end-to-end machine learning projects.

------

## Objectives

- Explore a real-world housing dataset.
- Build a Decision Tree Regression model.
- Generate predictions for unseen data.
- Evaluate model performance using Mean Absolute Error (MAE).
- Understand the effects of underfitting and overfitting.
- Learn the importance of model validation.

------

## Technologies Used

- Python
- pandas
- NumPy
- scikit-learn
- Jupyter Notebook

------

## Machine Learning Workflow

1. Load the dataset.
2. Select relevant features.
3. Define the prediction target.
4. Split the data into training and validation sets.
5. Train a Decision Tree Regressor.
6. Make predictions.
7. Evaluate the model using Mean Absolute Error.
8. Improve the model by tuning tree depth to reduce underfitting and overfitting.

------

## Results

The model demonstrates how Decision Trees can be used for regression problems and highlights why validation is essential when building machine learning models. By adjusting the tree depth, the project explores the trade-off between underfitting and overfitting.
